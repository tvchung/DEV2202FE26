DOM Element trong javascript

        Cú pháp tương tác: 
        1. tên thẻ html 
            document.getElementsByTagName("tag")
        2. thuộc tính id 
            document.getElementById("id name")
        3. thuộc tính class 
            document.getElementsByClassName('class name')
        4. css selector
            document.querySelectorAll('css selector')
