Review - Cấu trúc lặp

        1. for 
            * 4 dạng for khác nhau:
            - for đầy đủ:
                for (init; condition_expression ; step){
                    statement block;
                }
            - for khuyết:
                for( ; condition_expression ; step){
                    statement block;
                }
                for( init ;  ; step){
                    statement block;
                }
                
                for( init ;condition_expression  ; ){
                    statement block;
                }
            - for khuyết toàn bộ
                for(; ;){
                    statement block;
                }
            - for không thân:
                for(init, statement; condition_expression; step, statement; ){}

        2. while 
            - syntax:
            while(condition_expression){
                statement block;
            }
        3. do ... while
        - syntax:
            do{
                statement block;
            }while(condition_expression);
           
        4. foreach 
            - syntax:
            forEach(....);

        Ví dụ:
        Nhập số nguyên dương n;
        In ra và tính tổng các số nguyên tố
